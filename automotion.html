Although the original app allows users to view car listings, there is no option to save listings, making the browsing process tedious and inefficient. Thus, The first iteration of my prototype (on the left) not only allows users to access saved listings, but also provides suggested car listings based on a user's recent searches.

Face ID Sign-in

To make the app more efficient, users can now sign in with Apple Face ID.

Shop by Brand

A more intuitive filter feature displaying brand logos allows users to more easily and quickly find specific car dealers.

Shop by Car Type

A new feature allows users to filter their search by vehicle type.

Saved Listings and Detailed Vehicle Info

In order to offer a more personalized shopping experience, users will be able to save car listings and access them from the home page.

In addition, a more streamlined and consistent interface provides users with information on a vehicle's engine, seating capacity, and more.

Suggested Listings

As part of providing a personalized shopping experience, the home page provides suggested vehicle listings based on a user's recent searches and saved listings.
